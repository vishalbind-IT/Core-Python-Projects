# 💻 Core Python Projects

A collection of beginner-to-intermediate level Python projects developed for learning and practicing core Python concepts.

## 📂 Projects

### 🎓 Student Management System
- CLI-based system to manage student records (add, delete, search, update).
- Concepts: File handling, functions.

### 📝 To-Do List App
- Simple command-line app using lists and basic input.
- Concepts: Lists, user input, loops.

### ✊ Rock-Paper-Scissors Game
- Interactive CLI game vs computer.
- Concepts: `random` module, control flow.

### 📊 Monthly Expense Calculator
- Categorize and track expenses.
- Concepts: Loops, dictionaries, arithmetic logic.

### 🏦 Bank Account System (OOP)
- Simulates deposit/withdraw system.
- Concepts: Classes, objects, methods.
